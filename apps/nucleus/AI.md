## RMG Design System

The /design-system route at apps/nucleus/app/design-system/page.tsx
is a live showcase of the RMG component library. It demonstrates
the white-label capability of the Plato Suite — the same components
with RMG brand tokens applied.

Components live in packages/ui/components/rmg/. Token source is
packages/config/tokens/rmg.css. Do not add hardcoded hex values
to any component in this library.
